## Project Overview

Find-A-Lifer is an interactive web application that helps birders discover and plan trips to find "life birds" (species they've never seen before). The app combines eBird Status & Trends data with an interactive map to show where target species can be found by week.

## Technology Stack

- **Frontend**: React + TypeScript, Vite, MapLibre GL JS, Tailwind CSS
- **Backend**: Python 3.11+ with FastAPI
- **Data Storage**: IndexedDB (client-side) for user data, static JSON files for species/grid/weekly data
- **Map**: MapLibre GL JS with CARTO basemap tiles
