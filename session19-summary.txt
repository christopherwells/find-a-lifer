**Technical implementation details:**
- GoalBirdsTab uses `const { isSpeciesSeen } = useLifeList()` from LifeListContext.tsx
- Each species in the goal list checks `const seen = isSpeciesSeen(code)` when rendering
- Seen species: `line-through text-gray-400` for name + sci name, `bg-gray-100` background, green `checkmark Seen` badge
- Unseen species: normal `text-[#2C3E50]` styling and `bg-gray-50` background
- Species remain on the goal list regardless of seen status (NOT auto-removed per spec)
- When SpeciesTab calls `toggleSpecies()`, `setSeenSpecies()` updates shared context state
- This triggers immediate re-render of GoalBirdsTab - fully reactive without reload
- data-testid attributes: `goal-species-${code}`, `goal-species-seen-${code}`, `goal-species-unseen-${code}`
